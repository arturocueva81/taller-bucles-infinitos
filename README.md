# Taller — Bucles Infinitos en React

Proyecto educativo que muestra dos errores comunes en React que generan 
bucles infinitos y/o re-renders excesivos, junto con sus versiones corregidas.
Las funciones que generan los conflictos estan debidamente comentadas en el archivo index.html 
para evitar sobrecargar el browser debido a un desbordamiento de memoria.

---

## 🚀 Instalación y ejecución local

```bash
git clone https://github.com/arturocueva81/taller-bucles-infinitos.git
cd taller-bucles-infinitos
npm install
npm run dev
